# 🕌 Islamic Habit Tracker - Project Status

## 🌟 Current Overview
The Islamic Habit Tracker is now a comprehensive, production-ready spiritual companion. It combines habit tracking, Quran reading, daily remembrance, and gamification in a premium, dark-themed mobile experience.

## 🚀 Recent Accomplishments
### 🕋 Quran & Spirituality
- **Full Quran Library**: Access to all 114 Surahs with Arabic text and multi-language translations (English, Urdu, Hindi, French).
- **Jump-to-Ayah**: Smart navigation from bookmarks that auto-scrolls the reader to the specific verse.
- **Hasanat Rewards**: Reading a Surah now earns users dynamic points based on the number of Ayat.
- **Digital Tasbih**: A high-end Dhikr counter with haptic feedback and multiple phrase options.
- **Morning & Evening Azkar**: Proactive "Moment of Remembrance" screen that appears during sun-up and sun-down windows.

### 🔔 Smart Notifications
- **Daily Ayah**: Automated inspiring verse notifications scheduled for 8:00 AM every morning.
- **Habit Reminders**: Custom reminders for daily goals to ensure consistency.

### 👤 User Experience & Auth
- **Local Authentication**: Themed Login/Register flow with persistent local sessions.
- **Premium Profile**: A redesigned dashboard featuring a Rank Carousel (Servant to Muhsin) and live stat tracking.
- **Advanced Settings**: Control over translations, notifications, sound effects, and haptics.

## 🛠️ Architecture
- **State**: Zustand (Modular stores for Habits, Auth, Settings, Quran, Azkar).
- **Navigation**: React Navigation (Bottom Tabs + Full-screen Modals + Detail Stacks).
- **Storage**: AsyncStorage (Reliable, versioned storage service).
- **Services**: Custom API wrappers for Quran and local system services.
